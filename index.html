Apa Itu HTML ?

HTML atau HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan menyusun konten di web. HTML menyediakan struktur dasar untuk halaman web dengan menggunakan elemen-elemen yang ditandai dengan tag. Setiap elemen HTML dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut.

Elemen dan Tag HTML

Elemen HTML adalah bagian dari struktur halaman web. Elemen dapat berupa teks, gambar, video, audio, dan lain-lain. Setiap elemen memiliki tag yang digunakan untuk menandainya. Tag dapat berupa tag pembuka, tag penutup, dan tag tunggal

Contohnya!

<tag-pembuka> isinya </tag-penutup>

Tag penutup tidak harus selalu ada, terutama untuk tag-tag yang tidak memiliki konten teks.

Contohnya!

<tag-penutup/>

Apa Itu tag komentar ?

Tag komentar digunakan untuk menambahkan catatan atau keterangan pada kode HTML. Komentar tidak akan ditampilkan di halaman web, tetapi hanya bisa dilihat oleh pengembang web yang melihat kode HTML. Komentar biasanya digunakan untuk memberikan penjelasan tentang kode HTML, atau untuk menonaktifkan kode sementara.
